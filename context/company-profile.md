# TechCorp Company Profile

## Overview
TechCorp is a SaaS company headquartered in Karachi, Pakistan, providing enterprise project management and team collaboration software to businesses across South Asia and the Middle East.

## Flagship Product: TaskFlow Pro
TaskFlow Pro is a comprehensive project management platform designed for modern teams who need to move fast without losing alignment.

## Core Features
- **Task Management**: Create, assign, and track tasks with custom fields, due dates, priorities, and dependencies
- **Team Collaboration**: Real-time comments, @mentions, file attachments, and activity feeds
- **Project Reporting**: Gantt charts, burndown charts, velocity reports, and custom dashboards
- **Integrations**: Slack, Microsoft Teams, GitHub, GitLab, Jira import, Zapier, REST API
- **Time Tracking**: Built-in timer, manual entries, and exportable timesheets
- **Mobile Apps**: iOS and Android with full offline support
